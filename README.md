# Resume Builder

## Tech Stack

-  **TypeScript** (Language)
-  **MongoDB** (Database)
-  **Next.js** (Backend & Frontend SSR)
-  **NexAuth.js** (Auth Framework for Next.js)
-  **React** (Frontend)
-  **Tailwind** (Styling)

## Name Options

-  niceresu.me
-  buildaresu.me
-  thatsmyresu.me
-  myownresu.me
-  prettyresume.work

## How to Install

1. Install `node.js` and `yarn`
2. Install `mongodb`
3. Create a `.env.local` file with the current content

```
MONGODB_URL="mongodb://localhost:27017/resumebuilder"
```

5. run `yarn`
6. run `yarn dev`
